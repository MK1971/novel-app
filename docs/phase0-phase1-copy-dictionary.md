# Phase 0 / Phase 1 Copy Dictionary

This file is the baseline messaging guide for public reader-facing pages.

## Core model (canonical)

- Short: `Read Book 1 -> submit edits -> earn vote credits -> influence Book 2.`
- Public sentence: `In The Book With No Name, you submit contribution-backed edits. Each completed $2 contribution grants one vote credit for Peter Trull Solitary Detective.`

## Terminology rules

- Prefer: `contribution`, `contribution fee`, `vote credit`, `submit your version`.
- Avoid in marketing copy: `payment` as the primary noun, unless in legal/policy context.
- Keep legal accuracy: acceptance is reviewed and not guaranteed.

## Book naming standard

- `The Book With No Name (Collaborative Novel)`
- `Peter Trull Solitary Detective (Interactive Mystery)`

## CTA dictionary

- Primary manuscript CTA: `Enter the Manuscript`
- Primary auth CTA: `Become a Contributor (Sign Up)`
- Voting CTA: `Open voting hub`
- Feedback CTA: `Share ideas and feedback`

## CTA destination map

- `Enter the Manuscript` -> `/chapters`
- `Open voting hub` -> `/vote`
- `Share ideas and feedback` -> `/feedback`
- `Become a Contributor (Sign Up)` -> register modal

## Empty-state guidance

- No chapters live: use `Explore the Manuscript (Opening Soon)` and clarify staged releases.
- No public edits yet: explicitly state that accepted edits will appear here, then send user to `/chapters`.
- No leaderboard entries: explain how points are earned, then send user to `/chapters`.
