Place production-ready blog cover assets in this folder.

Expected source filenames:
- hero_banner.png
- twitter_post_2.png
- twitter_post_4.png

Generated 16:9 variants used by config/blog.php:
- blog-cards/hero_banner_16x9.png
- blog-cards/twitter_post_2_16x9.png
- blog-cards/twitter_post_4_16x9.png

These images are used automatically on blog index/show pages when present.
If a file is missing, the UI falls back to the post emoji cover.
